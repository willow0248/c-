2.#include<string.h>
  **sprintf**
  ` int sprintf(char *str, const char *format, ...);`
  参数说明
    参数	说明
    str	    指向目标字符数组的指针，存储格式化后的字符串
    format	格式控制字符串，包含普通字符和格式说明符
    ...	    可变参数列表，提供要格式化的值
    example: ` sprintf(buffer, "年龄: %d岁, 身高: %.1fcm", age, height);`
